# Инструкция по запуску приложения
---
* ### **Для начала клонируйте этот репозиторий к себе на PC коммандой** 
    ```bash
    git clone https://gitlab.crja72.ru/django/2024/autumn/course/students/185431-kiperfn-course-1187.git
    ```

* ### **Создайте виртуальное окружение**
    ##### Windows
    ```cmd
    python -m venv venv
    ```
    ##### Linux

    ```cmd
    sudo apt install python3-venv.
    ```

* ### **Активируем окружение коммандой**
    ##### Windows
    ```cmd
    venv\scripts\activate
    ```
    ###### Linux
    ```cmd
    source ./venv/bin/activate
    ```

* ### **Далее надо установить все зависимости**
    ```cmd
    pip install -r requirements.txt
    ```


* ### **Прописываем в консоль комманду**:
    ```bash
    python app2.py
    ```
